# Aeons: The Grand Unfolding

A web-based incremental game hosted on GitHub Pages.

🎮 **Play the game:** [Start Playing](https://joagwa.github.io/AeonsIdlePublic/)

## About

Aeons is an incremental idle game where you begin as a single particle of mass in a primordial universe. Absorb energy, convert dark matter, and progress through cosmic epochs from stellar nursery to advanced civilizations.

## Game Features

- **Infinite Universe** — Procedurally generated universe with quality-tiered motes
- **Progression System** — Upgrades that scale with investment
- **Idle Mechanics** — Passive generation and automated absorption
- **Offline Progress** — Earn resources even while away

## Getting Started

Just visit the link above and start playing! Your progress is automatically saved to your browser.
